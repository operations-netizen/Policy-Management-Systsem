## Installation Commands

### Install Frontend Dependencies
```bash
cd frontend
npm install
```

### Install Backend Dependencies
```bash
cd backend
npm install
```

### Install Database Dependencies
```bash
cd database
npm install
```

## Package.json Scripts

### Frontend (frontend/package.json)
```json
{
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview"
  }
}
```

### Backend (backend/package.json)
```json
{
  "scripts": {
    "dev": "NODE_ENV=development node --watch _core/index.js",
    "build": "esbuild _core/index.js --platform=node --packages=external --bundle --format=esm --outdir=dist --alias:@shared=./shared",
    "start": "NODE_ENV=production node dist/index.js",
    "test": "vitest run"
  }
}
```

## Environment Variables Required

### Backend (backend/.env)
```
MONGODB_URI=mongodb://localhost:27017/incentive-system
JWT_SECRET=your-secret-key-min-32-characters
PORT=3000
NODE_ENV=development
GHL_API_KEY=optional
GHL_LOCATION_ID=optional
GHL_WEBHOOK_SECRET=optional
```

### Frontend (frontend/.env)
```
VITE_OAUTH_PORTAL_URL=
VITE_APP_ID=
VITE_FRONTEND_FORGE_API_URL=
VITE_FRONTEND_FORGE_API_KEY=
VITE_ANALYTICS_ENDPOINT=
VITE_ANALYTICS_WEBSITE_ID=
```

```bash
nvm install 18

# Set default

```


## MongoDB Setup

### Local Installation
```bash
# macOS
brew tap mongodb/brew
brew install mongodb-community@6.0

# Ubuntu
sudo apt-get install mongodb-org

# Windows
Download from https://www.mongodb.com/try/download/community
```

### Cloud (MongoDB Atlas)
1. Create account at https://www.mongodb.com/cloud/atlas
2. Create free cluster
3. Get connection string
4. Update MONGODB_URI in backend/.env

## Troubleshooting

### Clear node_modules and reinstall (frontend)
```bash
cd frontend
rm -rf node_modules package-lock.json
npm install
```

### Clear node_modules and reinstall (backend)
```bash
cd backend
rm -rf node_modules package-lock.json
npm install
```

### Update all dependencies
```bash
npm update
```

### Check for outdated packages
```bash
npm outdated
```

### Audit for vulnerabilities
```bash
npm audit
npm audit fix
```

## Production Dependencies Only

### Install without dev dependencies (backend)
```bash
cd backend
npm install --production
```
